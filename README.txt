拼音测试小程序 - 使用说明
==============================

在电脑上使用：
1. 解压压缩包
2. 双击 pinyin_quiz.html 文件
3. 在浏览器中自动打开即可使用


在iPhone/iPad上使用（方法一 - 推荐）：
1. 将 pinyin_quiz.html 文件上传到任意免费网盘或文件托管服务
   - 例如：GitHub Pages、Netlify、Vercel等
2. 在iPhone的Safari浏览器中访问上传后的链接
3. 即可开始测试


在iPhone/iPad上使用（方法二）：
1. 在App Store下载"文件"应用（iOS自带）或第三方解压应用
2. 解压压缩包，找到 pinyin_quiz.html 文件
3. 在"文件"应用中点击HTML文件
4. 选择用Safari浏览器打开
5. 即可开始测试


在iPhone/iPad上使用（方法三）：
1. 将 pinyin_quiz.html 文件发送到iPhone（通过AirDrop、微信、邮件等）
2. 在iPhone上接收文件
3. 点击文件，选择用Safari浏览器打开
4. 即可开始测试


注意事项：
- 需要联网才能访问在线版本
- 本地HTML文件在iPhone上需要通过浏览器打开
- 建议使用方法一（上传到网络）获得最佳体验
